### _Instalation_
To install PW type:
```
git clone https://github.com/Glinek/PocketWhale.git
cd PocketWhale
pip install .
```
To run PW (I am working on making it run on one command):
```
cd ~/PocketWhale/PocketWhale
python PW.py
```
